## Getting Started

A backend API built with **Node.js**, **TypeScript**, **Express**, and **WebSocket**, using **MongoDB** for DB.

First, run the development server:

```bash
npm run dev
```

## **Tools**

- **Language:** TypeScript
- **Framework:** Express.js
- **Database:** MongoDB (via Mongoose)
- **WebSocket:** Socket.IO
- **Validation:** Joi
- **API Docs:** Swagger (OpenAPI)
